# Checklist

## 研究领域扩展
- [ ] 提取模板配置系统实现完成，支持动态加载模板
- [ ] 太阳能电池模板配置正确，能提取 PCE、Jsc、Voc、FF 参数
- [ ] 锂电池模板配置正确，能提取容量、循环稳定性、能量密度参数
- [ ] 传感器模板配置正确，能提取相关器件参数
- [ ] CLI 支持通过参数选择提取模板

## 桌面应用 - 进度显示
- [ ] 批量处理时显示当前文件名、已完成数量、总数量
- [ ] 单文件处理时显示当前处理阶段（解析/提取/获取影响因子）
- [ ] 进度信息实时更新，无卡顿

## 桌面应用 - 拖拽上传
- [ ] 拖拽区域视觉明显，有清晰的提示文字
- [ ] 拖拽文件时有视觉反馈效果
- [ ] 仅接受 PDF 文件，非 PDF 文件被拒绝
- [ ] 支持同时拖拽多个文件

## 桌面应用 - 结果预览和筛选
- [ ] 结果以表格形式展示，字段清晰
- [ ] 支持按列排序
- [ ] 筛选条件输入后实时过滤结果
- [ ] 表格支持横向滚动查看所有字段

## 桌面应用 - 导出格式选择
- [ ] 导出时提供格式选择弹窗
- [ ] 支持 Excel、JSON、CSV 格式导出
- [ ] 各格式导出数据完整正确

## 桌面应用 - 历史记录
- [ ] 历史记录列表显示处理时间、文件数量等关键信息
- [ ] 点击历史记录能正确加载历史结果
- [ ] 支持删除历史记录
- [ ] 历史记录持久化存储，重启应用不丢失

## Prompt 优化
- [ ] Prompt 模板结构化，层次清晰
- [ ] Few-shot 示例库包含各领域典型示例
- [ ] 提取准确率有明显提升（目标 90%+）

## 用户反馈修正
