## 1. 核心类

### 1.1 AnalysisPipeline

**位置**: `paperinsight/core/pipeline.py`

**描述**: 分析管线核心类，协调整个论文分析流程。

#### 类定义

```python
class AnalysisPipeline:
    """
    v3.0 分析管线
    
    整合 MinerU 解析 + 文本清洗 + LLM 提取的完整流程。
    """
    
    def __init__(
        self,
        output_dir: Union[str, Path],
        config: Optional[Dict[str, Any]] = None,
        cache_dir: Union[str, Path] = ".cache",
    ):
        """
        初始化分析管线
        
        Args:
            output_dir: 输出目录
            config: 完整配置字典（包含 mineru, llm, cleaner 等配置）
            cache_dir: 缓存目录
        """
```

#### 关键方法

##### process_pdf()

```python
def process_pdf(
    self,
    pdf_path: Path,
    max_pages: Optional[int] = None,
    use_cache: bool = True,
) -> Tuple[Optional[PaperData], Optional[Dict[str, Any]]]:
    """
    处理单个 PDF 文件（v3.0 流程）
    
    流程：
    1. 计算文件 MD5，检查缓存
    2. 使用 MinerU 解析 PDF -> Markdown
    3. 文本降噪，提取核心章节
    4. LLM 提取结构化数据
    5. Pydantic 校验
    
    Args:
        pdf_path: PDF 文件路径
        max_pages: 最大读取页数
        use_cache: 是否使用缓存
    
    Returns:
        (提取结果, 错误信息)
    """
```

##### run()

```python
def run(
    self,
    pdf_dir: Union[str, Path],
    recursive: bool = False,
    max_pages: Optional[int] = None,
    use_cache: bool = True,
    sort_by_if: bool = True,
    rename_pdfs: bool = False,
    rename_template: Optional[str] = None,
    pdf_files: Optional[List[Path]] = None,
    batch_size: int = 1,
) -> Dict[str, Any]:
    """
    运行分析管线
    
    Args:
        pdf_dir: PDF 目录
        recursive: 是否递归扫描
        max_pages: 最大读取页数
        use_cache: 是否使用缓存
        sort_by_if: 是否按影响因子排序
        rename_pdfs: 是否重命名 PDF
        rename_template: 重命名模板
        pdf_files: 指定 PDF 文件列表
        batch_size: 批处理大小
    
    Returns:
        运行统计信息
    """
```

#### 使用示例

```python
from paperinsight.core.pipeline import AnalysisPipeline

# 创建管线实例
pipeline = AnalysisPipeline(
    output_dir="./output",
    config={
        "llm": {"enabled": True, "provider": "openai", "api_key": "..."},
        "mineru": {"enabled": True, "mode": "api"},
    },
    cache_dir=".cache",
)

# 运行分析
stats = pipeline.run(
    pdf_dir="./pdfs",
    recursive=True,
    use_cache=True,
)

print(f"成功: {stats['success_count']}, 失败: {stats['error_count']}")
```

### 1.4 CacheManager

**位置**: `paperinsight/core/cache.py`

**描述**: 缓存管理器，处理解析和提取结果的缓存。

#### 关键方法

```python
class CacheManager:
    """缓存管理器"""
    
    def __init__(self, cache_dir: Union[str, Path] = ".cache"):
        """
        初始化缓存管理器
        
        Args:
            cache_dir: 缓存目录
        """
    
    def has_data_cache(self, md5: str) -> bool:
        """检查是否存在数据缓存"""
    
    def load_data_cache(self, md5: str) -> Optional[Dict[str, Any]]:
        """加载数据缓存"""
    
    def save_data_cache(self, md5: str, data: Dict[str, Any]) -> None:
        """保存数据缓存"""
    
    def has_markdown_cache(self, md5: str) -> bool:
        """检查是否存在 Markdown 缓存"""
    
    def load_markdown_cache(self, md5: str) -> Optional[str]:
        """加载 Markdown 缓存"""
    
    def save_markdown_cache(self, md5: str, markdown: str) -> None:
        """保存 Markdown 缓存"""
    
    def get_cache_stats(self) -> Dict[str, Any]:
        """获取缓存统计信息"""
    
    def clear_cache(self) -> None:
        """清理缓存"""
```

#### 使用示例

```python
from paperinsight.core.cache import CacheManager

# 创建缓存管理器
cache = CacheManager(cache_dir=".cache")

# 检查缓存
if cache.has_data_cache(md5):
    data = cache.load_data_cache(md5)
    print(f"从缓存加载: {data['paper_info']['title']}")
else:
    # 处理并保存缓存
    cache.save_data_cache(md5, paper_data.model_dump())

# 获取缓存统计
stats = cache.get_cache_stats()
print(f"缓存大小: {stats['total_size_mb']} MB")
```

from paperinsight.models.schemas import PaperData, PaperInfo, DeviceData

class BaseLLM(ABC):

    @abstractmethod

        self,

        temperature: float = 0.7,
        **kwargs,
    ) -> str:

    @abstractmethod
    def generate_json(
