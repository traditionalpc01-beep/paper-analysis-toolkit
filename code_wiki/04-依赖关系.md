# PaperInsight 依赖关系文档

本文档详细说明 PaperInsight 项目的依赖关系，包括 Python 依赖、Node.js 依赖和外部服务依赖。

---

## 1. Python 依赖

### 1.1 核心依赖

核心依赖是项目运行所必需的包，在 `requirements.txt` 和 `pyproject.toml` 中定义。

| 包名 | 版本要求 | 用途 | 模块 |
|------|----------|------|------|
| `typer` | >=0.9.0 | CLI 框架 | cli.py |
| `rich` | >=13.0.0 | 终端美化输出 | cli.py, terminal.py |
| `PyMuPDF` | >=1.23.0 | PDF 处理 | parser/, pdf_utils.py |
| `pdfplumber` | >=0.7.0 | PDF 文本提取 | parser/, pdf_utils.py |
| `pandas` | >=2.0.0 | 数据处理 | reporter.py |
| `openpyxl` | >=3.1.0 | Excel 文件操作 | reporter.py |
| `pydantic` | >=2.0.0 | 数据验证 | models/schemas.py |
| `pyyaml` | >=6.0 | 配置文件解析 | utils/config.py |
| `requests` | >=2.31.0 | HTTP 请求 | web/ |
| `beautifulsoup4` | >=4.12.0 | HTML 解析 | web/ |
| `tqdm` | >=4.66.0 | 进度条 | pipeline.py |
| `python-dotenv` | >=1.0.0 | 环境变量管理 | config.py |

## 3. 外部服务依赖

### 3.1 LLM 服务

项目支持多个 LLM 提供商，需要配置相应的 API Key。

| 提供商 | 配置项 | 获取方式 |
|--------|--------|----------|
| OpenAI | `llm.api_key` | https://platform.openai.com/api-keys |
| DeepSeek | `llm.api_key` | https://platform.deepseek.com/ |
| 文心一言 | `llm.wenxin.client_id`, `llm.wenxin.client_secret` | https://console.bce.baidu.com/qianfan/ |
| Longcat | `llm.api_key` | https://api.longcat.chat/ |

#### 配置示例

```yaml
llm:
  enabled: true
  provider: openai  # openai, deepseek, wenxin, longcat
  api_key: sk-xxxxx
  model: gpt-4o
  timeout: 120
```

### 3.2 PDF 解析服务

#### MinerU

| 模式 | 配置项 | 说明 |
|------|--------|------|
| API | `mineru.token` | 云端 API Token |
| CLI | `mineru.cli_path` | 本地 CLI 路径 |

**获取 API Token**: https://mineru.net/

```yaml
mineru:
  enabled: true
  mode: api  # api 或 cli
  token: your-token-here
```

### 3.3 OCR 服务

#### PaddleX API

```yaml
paddlex:
  enabled: true
  token: your-paddlex-token
```

### 3.4 期刊数据服务

#### MJL (Master Journal List)

- **用途**: 期刊元数据解析
- **API**: https://mjl.clarivate.com/
- **配置**: 无需 API Key，直接使用

#### LetPub

- **用途**: 影响因子获取
- **网站**: https://www.letpub.com.cn/
- **配置**: 无需 API Key，爬虫方式

#### Web of Science API

- **用途**: 官方期刊数据
- **配置**: 需要 API Key

```yaml
web_search:
  web_of_science:
    enabled: true
    api_key: your-wos-api-key
```

#### AI 模型影响因子获取（实验性）

```yaml
web_search:
  ai_model_if:
    enabled: true
    qianwen_api_key: sk-xxxxx  # 通义千问
    kimi_api_key: sk-xxxxx     # Kimi
```

## 6. 依赖冲突解决

### 6.1 常见冲突

#### PyMuPDF 与其他 PDF 库

```bash
# 如果遇到冲突，优先使用 PyMuPDF
pip uninstall pdfplumber
pip install PyMuPDF
```

#### PaddlePaddle GPU 版本

```bash
# CPU 版本
pip install paddlepaddle

# GPU 版本（需要 CUDA 11.x）
pip install paddlepaddle-gpu
```

### 6.2 虚拟环境推荐

```bash
# 使用 venv
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate     # Windows

# 或使用 conda
conda create -n paperinsight python=3.10
conda activate paperinsight
```

## 9. Docker 环境依赖

### Dockerfile 示例

```dockerfile
FROM python:3.10-slim

WORKDIR /app

# 安装系统依赖
RUN apt-get update && apt-get install -y \
    libgl1-mesa-glx \
    libglib2.0-0 \
    && rm -rf /var/lib/apt/lists/*

# 安装 Python 依赖
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# 安装项目
COPY . .
RUN pip install -e .

ENTRYPOINT ["paperinsight"]
```
