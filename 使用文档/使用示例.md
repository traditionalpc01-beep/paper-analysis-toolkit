## 场景二：批量处理

### 任务：分析整个文件夹的PDF（50+篇）

提示：如果PDF分散在多个子目录中，可以直接使用 `--recursive` 递归扫描：

```bash
python 脚本/analyze_papers.py --pdf-dir /your/path/to/pdfs --recursive
```

1. **创建批处理脚本**

   创建 `batch_process.py`:
   ```python
   import os
   from pathlib import Path
   import subprocess
   
   # PDF目录
   pdf_dir = Path("../示例数据/pdfs")
   pdf_files = list(pdf_dir.glob("*.pdf"))
   
   # 分批，每批20个
   batch_size = 20
   
   for i in range(0, len(pdf_files), batch_size):
       batch = pdf_files[i:i+batch_size]
       
       # 创建临时目录
       temp_dir = Path(f"../示例数据/batch_{i//batch_size}")
       temp_dir.mkdir(exist_ok=True)
       
       # 复制文件
       for pdf in batch:
           os.link(pdf, temp_dir / pdf.name)
       
       # 修改配置并运行
       print(f"\n处理批次 {i//batch_size + 1}")
       # 运行分析脚本...
   ```

2. **运行批处理**
   ```bash
   python batch_process.py
   ```

## 场景四：数据统计

### 任务：生成统计报告

创建 `generate_statistics.py`:

```python
import pandas as pd
import matplotlib.pyplot as plt

# 读取Excel结果
df = pd.read_excel('../输出结果/论文分析报告_xxx.xlsx')

# 统计1：期刊分布
journal_counts = df['期刊名称'].value_counts()
print("\n期刊分布:")
print(journal_counts)

# 统计2：EQE分布
eqe_values = df['EQE（外量子效率）'].dropna()
eqe_numeric = eqe_values.str.extract(r'(\d+\.?\d*)')[0].astype(float)

print(f"\nEQE统计:")
print(f"  最高: {eqe_numeric.max():.2f}%")
print(f"  平均: {eqe_numeric.mean():.2f}%")
print(f"  中位数: {eqe_numeric.median():.2f}%")

# 绘制图表
fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# 期刊分布图
journal_counts.head(10).plot(kind='bar', ax=axes[0])
axes[0].set_title('论文期刊分布 (Top 10)')
axes[0].set_xlabel('期刊')
axes[0].set_ylabel('论文数量')

# EQE分布图
eqe_numeric.hist(bins=20, ax=axes[1])
axes[1].set_title('EQE值分布')
axes[1].set_xlabel('EQE (%)')
axes[1].set_ylabel('论文数量')

plt.tight_layout()
plt.savefig('../输出结果/统计图表.png')
print("\n图表已保存: 统计图表.png")
```

## 场景五：导出为其他格式

### 导出为JSON

脚本已内置可选导出，无需再手动读取 Excel：

```bash
python 脚本/analyze_papers.py --export-json --export-csv
```

同时也支持导出“缺失项报告”（逐篇标注哪些字段缺失，以及关键指标缺失原因）：

```bash
python 脚本/analyze_papers.py --export-md --export-docx
```

```python
import json

# 读取Excel
df = pd.read_excel('论文分析报告.xlsx')
```

### 导出为Markdown表格

```python
def export_to_markdown(df, output_file):
    """导出为Markdown表格"""
    md_content = "# 论文分析报告\n\n"
    md_content += "## 统计信息\n\n"
    md_content += f"- 总论文数: {len(df)}\n"
    md_content += f"- 平均影响因子: {df['影响因子'].mean():.2f}\n\n"
    
    md_content += "## 论文列表\n\n"
    
    # 选择关键列
    key_columns = ['期刊名称', '影响因子', '论文标题', 'EQE（外量子效率）']
    df_display = df[key_columns]
    
    # 转换为Markdown表格
    md_content += df_display.to_markdown(index=False)
    
    with open(output_file, 'w', encoding='utf-8') as f:
        f.write(md_content)
    
    print(f"Markdown报告已保存: {output_file}")
```

## 场景六：定期自动分析

### 使用cron定时任务（Linux/Mac）

1. **创建定时脚本**
   ```bash
   # 编辑crontab
   crontab -e
   ```

2. **添加定时任务**
   ```cron
   # 每周一早上8点运行
   0 8 * * 1 cd /path/to/论文分析工具包 && python3 脚本/analyze_papers.py
   ```

### 使用Windows任务计划

1. 打开"任务计划程序"
2. 创建基本任务
3. 设置触发器：每周
4. 操作：启动程序
   - 程序：`python.exe`
   - 参数：`脚本\analyze_papers.py`
   - 起始位置：`C:\path\to\论文分析工具包`
