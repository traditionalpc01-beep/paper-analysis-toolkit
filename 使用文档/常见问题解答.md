## 安装和配置问题

### Q1: pip install 失败怎么办？

**问题现象：**
```
ERROR: Could not find a version that satisfies the requirement pdfplumber
```

**解决方案：**
1. 更新pip版本：
   ```bash
   python -m pip install --upgrade pip
   ```

2. 使用国内镜像源：
   ```bash
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

3. 单独安装失败的包：
   ```bash
   pip install pdfplumber
   pip install openpyxl
   pip install python-docx
   ```

### Q2: 提示"Permission denied"怎么办？

**解决方案：**
1. 使用用户级安装：
   ```bash
   pip install --user -r requirements.txt
   ```

2. 或使用虚拟环境：
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   # 或
   venv\Scripts\activate     # Windows
   pip install -r requirements.txt
   ```

### Q3: Python版本不兼容？

**解决方案：**
确保使用Python 3.8或更高版本：
```bash
python --version
# 如版本过低，安装新版本Python
```

## 运行错误

### Q4: 找不到PDF文件？

**问题现象：**
```
错误: 未找到PDF文件！
请将PDF文件放入: /path/to/示例数据/pdfs
```

**解决方案：**
1. 确认目录存在：
   ```bash
   ls -la 示例数据/pdfs/
   ```

2. 确认文件扩展名正确（必须是小写的 `.pdf`）

3. 使用命令行参数指定实际目录：
   ```bash
   python 脚本/analyze_papers.py --pdf-dir /your/actual/path/to/pdfs
   ```

### Q5: PDF文件无法打开？

**问题现象：**
```
错误:PdfReadError: EOF marker not found
```

**解决方案：**
1. 检查PDF文件是否损坏：
   ```bash
   # 尝试用其他工具打开PDF
   ```

2. 使用PDF修复工具修复损坏的文件

3. 跳过损坏文件：
   ```python
   try:
       text = extract_full_text(pdf_path)
   except:
       print(f"跳过损坏文件: {pdf_path.name}")
       continue
   ```

### Q6: 内存不足错误？

**问题现象：**
```
MemoryError: Unable to allocate array
```

**解决方案：**
1. 分批处理文件（见高级配置说明）

2. 处理完每个文件后清理内存：
   ```python
   import gc
   gc.collect()
   ```

3. 减少一次性加载的PDF页数

## 数据提取问题

### Q7: 期刊显示"未知期刊"？

**原因分析：**
- 期刊不在预设数据库中
- PDF格式不标准，无法识别期刊名

**解决方案：**
1. 在 `配置文件/journal_impact_factors.json` 中添加期刊：
   ```json
   {
     "自定义期刊": {
       "新期刊名称": {
         "impact_factor": 10.5,
         "category": "Custom"
       }
     }
   }
   ```

2. 检查PDF第一页的期刊信息格式

3. 修改期刊识别模式：
   ```python
   journal_patterns = [
       # 添加新模式
       r'New Journal Name',
   ]
   ```

### Q8: EQE等数据提取不到？

**原因分析：**
- 文章类型不匹配（综述、基础研究）
- 数据格式不在预设模式中
- PDF是扫描版（无文本层）

**解决方案：**
1. 确认文章类型是否为器件研究

2. 检查PDF是否有文本层：
   ```bash
   # 使用pdfplumber测试
   python -c "import pdfplumber; pdf=pdfplumber.open('file.pdf'); print(pdf.pages[0].extract_text())"
   ```

3. 添加自定义提取模式：
   ```python
   patterns = [
       # 添加新模式
       r'新格式模式',
   ]
   ```

### Q9: 提取的标题不正确？

   def extract_title(text, filename):

           parts = filename.split(' - ')
           if len(parts) >= 4:
               return parts[3].replace('.pdf', '')

2. 使用PDF的元数据（如果有）：
   ```python
   import PyPDF2
   reader = PyPDF2.PdfReader(pdf_path)
   if reader.metadata.title:
       return reader.metadata.title
   ```

### Q15: 如何处理大量PDF（>100个）？

**解决方案：**
```python
# 分批处理
batch_size = 50
for i in range(0, len(pdf_files), batch_size):
    batch = pdf_files[i:i+batch_size]
    results = process_batch(batch)
    save_intermediate(results, i)
```

### Q18: 如何批量更新影响因子？

**解决方案：**
创建更新脚本：
```python
def update_impact_factors(excel_file):
    wb = openpyxl.load_workbook(excel_file)
    ws = wb.active
    
    for row in range(2, ws.max_row + 1):
        journal = ws.cell(row=row, column=3).value
        if journal in JOURNAL_IMPACT_FACTORS:
            ws.cell(row=row, column=4).value = JOURNAL_IMPACT_FACTORS[journal]
    
    wb.save(excel_file)
```
