# 快速开始指南

## 5分钟上手教程

### 第一步：环境准备

1. **确认Python版本**
   ```bash
   python --version
   # 建议使用 Python 3.8 或更高版本
   ```

2. **安装依赖包**
   ```bash
   cd 论文分析工具包
   pip install -r requirements.txt
   ```

### 第二步：准备PDF文件

1. 创建PDF目录（如果不存在）
   ```bash
   mkdir -p 示例数据/pdfs
   ```

2. 将您的PDF论文文件复制到该目录
   ```bash
   # 示例：复制文件
   cp /path/to/your/papers/*.pdf 示例数据/pdfs/
   ```

### 第三步：运行分析

```bash
python 脚本/analyze_papers.py
```

如果系统提示找不到 `python` 命令（Mac/Linux 常见），请改用：

```bash
python3 脚本/analyze_papers.py
```

可选参数（常用）：

```bash
# 递归扫描子目录中的PDF
python 脚本/analyze_papers.py --recursive

# 每篇最多读取前5页（加速，可能降低命中率）
python 脚本/analyze_papers.py --max-pages 5

# 同时导出 JSON / CSV
python 脚本/analyze_papers.py --export-json --export-csv

# 同时导出缺失项报告（Markdown / Word）
python 脚本/analyze_papers.py --export-md --export-docx
```

### 第四步：查看结果

分析完成后，在 `输出结果/` 目录下找到生成的Excel文件。

## 典型使用流程

```
准备PDF → 运行脚本 → 查看Excel → 数据分析
   ↓           ↓           ↓          ↓
  放入     自动提取    生成报告   按IF排序
  目录     关键信息    (排序后)
```

## 常见问题

**Q: 没有找到PDF文件？**  
A: 确认PDF文件已放入 `示例数据/pdfs/` 目录，且文件扩展名为 `.pdf` / `.PDF`

**Q: 期刊显示"未知期刊"？**  
A: 该期刊可能不在预设的期刊数据库中，可在 `配置文件/journal_impact_factors.json` 中添加

**Q: EQE等数据未提取到？**  
A: 可能原因：
- 文章类型不是器件研究（如综述、基础研究）
- PDF是扫描版（无文本层）
- 数据格式不在预设模式中
