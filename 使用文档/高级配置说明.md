## 性能优化

### 批量处理大量文件

如果需要处理数百个PDF文件：

```python
# 方法1：分批处理
def process_in_batches(pdf_files, batch_size=50):
    for i in range(0, len(pdf_files), batch_size):
        batch = pdf_files[i:i+batch_size]
        # 处理这一批
        results = []
        for pdf_file in batch:
            result = analyze_pdf(pdf_file)
            if result:
                results.append(result)
        
        # 保存中间结果
        save_intermediate_results(results, i)
```

### 并行处理（多核CPU）

```python
from multiprocessing import Pool

def analyze_pdf_wrapper(pdf_path):
    return analyze_pdf(pdf_path)

# 使用多进程
with Pool(processes=4) as pool:  # 使用4个CPU核心
    results = pool.map(analyze_pdf_wrapper, pdf_files)
```

### 内存优化

```python
# 处理完每个PDF后释放内存
import gc

def analyze_pdf(pdf_path):
    text = extract_full_text(pdf_path)
    result = extract_all_info(text)
    
    # 释放大文本变量
    del text
    gc.collect()
    
    return result
```

## 配置文件管理

### 创建用户配置

创建 `config.py`：

```python
# config.py

# PDF目录
PDF_INPUT_DIR = "./示例数据/pdfs"

# 输出目录
OUTPUT_DIR = "./输出结果"

# 是否启用并行处理
ENABLE_PARALLEL = True

# 并行进程数
NUM_PROCESSES = 4

# 输出格式
OUTPUT_FORMAT = 'xlsx'  # 可选: xlsx, json, csv
```

然后在主脚本中导入：

```python
import sys
sys.path.append('../配置文件')
from config import *
```

### 使用命令行参数覆盖默认配置

```bash
python 脚本/analyze_papers.py \
  --pdf-dir /path/to/pdfs \
  --output-dir /path/to/output \
  --journal-config /path/to/journals.json
```
