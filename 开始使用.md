## 🚀 快速开始（3步）

### 第一步：安装依赖

打开终端/命令行，进入工具包目录：

```bash
cd 论文分析工具包
pip install -r requirements.txt
```

### 第二步：准备PDF文件

将您的PDF论文文件复制到：

```
论文分析工具包/示例数据/pdfs/
```

### 第三步：运行分析

**方法一：使用快速启动脚本（推荐新手）**

Windows:
```bash
双击运行 start.bat
```

Linux/Mac:
```bash
chmod +x start.sh
./start.sh
```

**方法二：直接运行脚本**

```bash
python 脚本/analyze_papers.py
```

## 🔧 自定义扩展

工具包支持多种自定义：

### 添加新期刊

编辑 `配置文件/journal_impact_factors.json`，添加：
```json
{
  "期刊名称": {
    "impact_factor": 影响因子数值,
    "category": "学科分类"
  }
}
```

### 修改提取规则

编辑 `脚本/analyze_papers.py`，修改正则表达式模式。

### 调整输出格式

在 `create_excel_report()` 函数中修改样式和布局。
