# 推送工具包到GitHub私有仓库

## 方法一: 使用GitHub网页创建仓库(推荐)

### 步骤1: 在GitHub上创建私有仓库

1. 打开浏览器,访问 https://github.com/new
2. 填写仓库信息:
   - Repository name: `paper-analysis-toolkit`
   - Description: `PDF论文分析工具包 - 自动提取QLED器件性能数据并生成报告`
   - 选择 **Private** (私有仓库)
   - **不要**勾选 "Add a README file"
   - **不要**勾选 "Add .gitignore"
   - 点击 "Create repository"

### 步骤2: 在本地添加远程仓库并推送

创建好仓库后,GitHub会显示推送命令。您需要:

```bash
cd "/Users/locybe/论文处理/论文分析工具包"

# 添加远程仓库 (替换 YOUR_USERNAME 为您的GitHub用户名)
git remote add origin https://github.com/YOUR_USERNAME/paper-analysis-toolkit.git

# 推送代码到GitHub
git branch -M main
git push -u origin main
```

## 方法二: 使用GitHub CLI (如果已安装gh命令)

```bash
# 创建私有仓库并推送
cd "/Users/locybe/论文处理/论文分析工具包"
gh repo create paper-analysis-toolkit --private --source=. --remote=origin --push
```

## 方法三: 使用Personal Access Token

### 1. 创建GitHub Token

1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 `repo` 权限
4. 生成并复制token

### 2. 使用Token创建仓库

```bash
# 设置token环境变量
export GITHUB_TOKEN="your_token_here"

# 创建仓库
curl -X POST \
  -H "Authorization: token $GITHUB_TOKEN" \
  -H "Accept: application/vnd.github.v3+json" \
  https://api.github.com/user/repos \
  -d '{"name":"paper-analysis-toolkit","description":"PDF论文分析工具包","private":true}'

# 添加远程仓库并推送
cd "/Users/locybe/论文处理/论文分析工具包"
git remote add origin https://github.com/YOUR_USERNAME/paper-analysis-toolkit.git
git push -u origin main
```
