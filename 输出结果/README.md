## 🚀 下次使用

### 方法1: 直接运行

```bash
cd "/Users/locybe/论文处理/论文分析工具包"
python3 脚本/analyze_papers.py --pdf-dir "/path/to/your/pdfs" --output-dir "./输出结果"
```

### 方法2: 使用启动脚本

```bash
cd "/Users/locybe/论文处理/论文分析工具包"
./start.sh
```

### 方法3: 自定义参数

```bash
# 只读取前10页（加速）
python3 脚本/analyze_papers.py --pdf-dir "./pdfs" --max-pages 10

# 导出JSON和CSV
python3 脚本/analyze_papers.py --pdf-dir "./pdfs" --export-json --export-csv

# 导出缺失项报告（Markdown / Word）
python3 脚本/analyze_papers.py --pdf-dir "./pdfs" --export-md --export-docx

# 递归扫描子目录
python3 脚本/analyze_papers.py --pdf-dir "./pdfs" --recursive
```

---

## 📝 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--pdf-dir` | PDF文件目录 | ./示例数据/pdfs |
| `--output-dir` | 输出目录 | ./输出结果 |
| `--journal-config` | 期刊配置文件 | ./配置文件/journal_impact_factors.json |
| `--recursive` | 递归扫描子目录 | False |
| `--max-pages` | 每篇论文最大页数 | 0 (不限制) |
| `--export-json` | 导出JSON | False |
| `--export-csv` | 导出CSV | False |
| `--export-md` | 导出Markdown缺失项报告 | False |
| `--export-docx` | 导出Word缺失项报告 | False |
| `--no-json` | 不使用期刊配置 | False |

## 🔧 扩展功能

### 添加新期刊

编辑 `配置文件/journal_impact_factors.json`:

```json
{
  "期刊名称": {
    "impact_factor": 15.0,
    "category": "材料科学"
  }
}
```

### 自定义提取规则

编辑 `脚本/analyze_papers.py` 中的提取函数:

- `extract_eqe_comprehensive()` - EQE提取
- `extract_lifetime_comprehensive()` - 寿命提取
- `extract_cie_comprehensive()` - CIE坐标提取
- `extract_device_structure()` - 器件结构提取
