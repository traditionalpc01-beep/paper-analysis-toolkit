# 输出结果目录

此目录用于存放分析结果。

## 输出文件

运行分析脚本后，将在此目录生成以下文件：

1. **Excel报告** - `论文分析报告_时间戳.xlsx`
   - 包含所有提取的信息
   - 按影响因子排序
   - 可直接用Excel或WPS打开

## 文件格式

Excel文件包含以下列：

| 列序 | 列名 | 说明 |
|------|------|------|
| A | File | PDF文件名 |
| B | URL | 文件路径 |
| C | 期刊名称 | 发表的期刊 |
| D | 影响因子 | 期刊IF值 |
| E | 作者 | 前3位作者 |
| F | 论文标题 | 论文标题 |
| G | 器件结构 | 器件各层材料 |
| H | 优化层级 | 优化层面 |
| I | 优化策略 | 具体优化方法 |
| J | EQE | 外量子效率 |
| K | 色度坐标 | CIE坐标 |
| L | 寿命 | T50或lifetime |
| M | 补充信息 | 波长、亮度等 |

## 使用建议

1. 查看前10行（高影响因子论文）
2. 关注EQE列，筛选高性能器件
3. 对比不同优化策略的效果
4. 保存结果用于后续分析
